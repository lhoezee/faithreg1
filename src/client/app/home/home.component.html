This extended class will offer further in-depth discussions and training on using the latest web development technologies preparing students to enter into the professional work field of web development.

When we meet:

We will meet in the computer lab directly after the Faith Homeschool classes are dismissed from the auditorium. We anticipate our class to be about 1 hour to 1.5 hours.

Note: Because this will be right around lunch time, the student may want to bring a small lunch or snack.

Price:

There will be no cost for this extra class.

What we will cover:

  • Beginner through advanced level Javascript
  • Web frameworks such as Bootstrap and AngularJS which are used by large enterprise companies to build their online web applications.
  • Learn how to use version control systems such as Git. This allows multiple developers to work on the same project and control source code versions.
  • Learn how to create web forms and save information to a backend server, such as the registration page on this site.
  • Occasionally we may work on real world projects such as banking applications and more.
  • Many more items that come with day-to-day development practices

The goal of the class:

The goal of this extended class is to go beyond the basics of the intro level class. This allows the student to see the big picture of the entire web development process.

Who can join:

Any student in 10th, 11th, or 12th grade. We ask the students should be serious about this as we will be fully focused on their success and education.
